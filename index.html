MERN Stack Team Project
- GameDart -

Fullstack project made in collaboration with a group of developers from my cohort at App/Academy. Utilizes MERN stack technologies: MongoDB, Express, React.js and Node. A site where you can add, remove and manage games in your personal backlog. Users can also be suggested other games similar to their backlog, as well as discover new ones through our search algorithm.

Personal JavaScript Project
- Granblue Versus Frame Tool -

Frame data tool made for the competitive Granblue Fantasy: Versus community. The goal behind this project was to create an accessible resource hub for competitors to easily access accurate frame data of each characters moveset, allowing them to easily break down micro-moments in-game more accurately and gain a further understanding of the game quickly.
